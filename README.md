# email-test-node-ang
1.Parse the folder
2.Run the npm install or pnpm install at both frontend and backend folders
3.Replace Sender Email id at this file (email-test-node-ang\backend\otpModule.cjs).
4.Replace To Email id at this file (email-test-node-ang\backend\otpModule.cjs).
5.Key in Sender Password at this file (email-test-node-ang\backend\otpModule.cjs).
6.Backend terminal - Run nodemon app.js
7.Frontend terminal - Run grunt (if grunt doesn't work for you install grunt needed modules)
8.Check OTP - has been implemented without database, so expect that checking at front end.
